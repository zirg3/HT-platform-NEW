# v0.04-1 · Еженедельное расписание и ссылка на урок

**Ветка (план):** `dev-0.4` от `dev`

## Добавлено

- Миграция `20260520120000_lesson_meeting_url.sql` — колонка `lessons.meeting_url`
- Поле «Ссылка для подключения» в форме урока; блок в карточке урока (ученик и преподаватель)
- `safeLessonMeetingHref` — безопасные префиксы для `href`
- Чекбокс «Каждую неделю» при **создании** урока + дата «до» включительно
- Серия записей в `lessons` с одним `recurrence_group_id`, до 52 экземпляров
- `buildWeeklyStartsUtc`, `fetchLessonsBetween`, предупреждения о пересечениях по всей серии

## Поведение

- Разовый урок — без галочки
- С галочкой — слоты по тем же дню недели и времени каждую неделю до выбранной даты
- Редактирование серии целиком не реализовано (как отдельные уроки)
