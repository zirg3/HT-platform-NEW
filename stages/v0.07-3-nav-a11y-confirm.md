# v0.07-3 — навигация, a11y, подтверждения

## Что сделано

- **AppShell**: постоянная навигация по роли (`RoleNav`), skip-link «Перейти к содержимому», `id="main-content"`.
- **NavLink** + `lib/auth/nav-links.ts`: единый стиль ссылок, `aria-current="page"`.
- **Страницы**: заголовок `h1` у ученика; убраны дублирующие кнопки «назад» (навигация в шапке).
- **AdminNav** удалён — те же пункты в шапке для админа.
- **Календарь**: слоты без tab-stop (`tabIndex={-1}`); навигация недели через `Link` + `buttonVariants`; подсказка прокрутки на мобильном.
- **Админка**: `confirm()` перед удалением пользователя / курса / привязки; фильтры ролей с `aria-current`.
- **Диалог**: «Закрыть»; форма урока — `grid-cols-1 sm:grid-cols-2`, `aria-busy` на сохранении.
