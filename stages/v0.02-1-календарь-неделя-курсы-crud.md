# v0.02-1 · Календарь (неделя), CRUD уроков, курсы

**Ветка (план):** `dev-0.2` от `dev`

## Добавлено

- `date-fns` / `date-fns-tz` — неделя, TZ `Europe/Moscow`
- `lib/schedule/*` — запросы, права, конфликты, даты
- `app/actions/schedule.ts`, `app/actions/courses.ts`
- UI: `WeekCalendar`, `LessonDialog`, `LessonBlock`, `SchedulePage`
- `/admin/courses` — справочник курсов (CRUD)
- ЛК: `/teacher`, `/student`, `/manager`, `/admin` — недельное расписание
- Миграции: отмена урока учеником, insert урока для staff; RLS уроков — в `initial_schema` (без отдельного fix-файла)
- `scripts/new-dev-branch.ps1`, `docs/BRANCHING.md`

## Поведение

- Неделя пн–вс, 08:00–22:00, навигация `?week=YYYY-MM-DD`
- Клик по слоту — создание (преподаватель / staff)
- Отмена: статус `cancelled`, бейдж «Отменён», зачёркивание
- Предупреждение при пересечении (не блокирует сохранение)

## Проверка

1. Миграции в Supabase (включая `20260517130000`, `20260517130100`)
2. Админ: `/admin/courses` — добавить курсы
3. Преподаватель: `/teacher` — создать урок
4. Ученик: `/student` — просмотр и отмена
