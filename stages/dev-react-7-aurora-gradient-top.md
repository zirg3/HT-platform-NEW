# dev-react-7 — фон aurora без белого верха

## Изменения

- **`globals.css`**: `--background` чуть темнее (`0.935` вместо `0.97`).
- **`.aurora-bg::before`**:
  - убраны яркие radial-bloom в верхних углах (`-5% -5%`);
  - верх — мягкий однородный лавандовый fade без белой вспышки;
  - тонкая горизонтальная светлая линия ~50% высоты;
  - тонкая диагональная полоса через центр;
  - цветовые пятна смещены к бокам (58–62%) и низу.

## Проверка

`npm run dev` → верх экрана без ослепляющего белого; акцент — тонкая полоса по центру.
