# v1.2-5 — вход через браузерный Supabase

## Проблема
«Ошибка соединения» — падал вызов Server Action `loginAction`, не сам Supabase.

## Решение
- `login-form.tsx`: `signInWithPassword` через `createBrowserClient` (cookies в браузере)
- `public-env.ts`: проверка URL/ключа без пробелов
- `fetch.ts`: keepalive только в браузере; `AbortSignal.any` для таймаута

- После signIn: `resolveLoginRoleAction` на сервере (retry + `app_metadata`)
- Миграция backfill `20260522160000_backfill_profiles.sql`
