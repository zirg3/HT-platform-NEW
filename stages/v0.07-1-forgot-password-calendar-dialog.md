# v0.07-1 — восстановление пароля, календарь, фон диалога

## Что сделано

### Восстановление пароля
- На `/login` ссылка **«Забыли пароль?»**
- `/login/forgot-password` — запрос письма (`resetPasswordForEmail`)
- `/login/reset-password` — новый пароль после перехода по ссылке из письма
- `lib/site-url.ts`, `NEXT_PUBLIC_SITE_URL` в `.env.example`
- В Supabase Dashboard → Auth → URL Configuration добавить redirect:  
  `{SITE_URL}/auth/callback?next=/login/reset-password`

### Календарь
- `WeekDatePicker` — по центру шапки: иконка календаря + диапазон недели, клик открывает выбор даты и переход к нужной неделе

### UX диалога урока
- `DialogContent`: `bg-background` + `shadow-lg` вместо прозрачного `bg-popover`
- Селекты в форме урока: `bg-background`
