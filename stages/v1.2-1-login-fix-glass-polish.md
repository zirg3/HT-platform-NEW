# v1.2-1 — вход и полировка Glass Aurora

## Вход
- `login-form.tsx`: убран `useActionState` → прямой вызов `loginAction` + `useTransition` (нет ошибки «unexpected response»)
- Чтение email/пароля из DOM перед отправкой — исправлена гонка при быстром Enter
- Защита от двойной отправки (`isSubmittingRef`)
- Полноэкранный оверлей «Вход выполнен…» при редиректе

## Расписание (макет Glass Aurora)
- Нижняя сетка **3 колонки**: предстоящие | прошедшие | статистика
- Карточки agenda: слот, «Через N ч», тема, «Онлайн»
- Календарь: время диапазоном, имя + предмет
- Topbar: профиль справа, колокольчик
- Sidebar: иконка Sparkles, блок «Нужна помощь?»
- Пастельные цвета статусов (mint / lavender / peach / rose)
- Усилен aurora-bg и glass-panel
