# v1.2 — Glass Aurora (оболочка UI)

## Тема
- Aurora mesh-фон, glass-панели, фиолетово-бирюзовый primary
- Видимые обводки полей (`.glass-input`, `--border`)

## Layout (как макет)
- Сайдбар: навигация + «Создать занятие»
- Шапка: аватар-заглушка, имя, роль, выход
- Страница: заголовок → календарь в glass → сетка (списки | статистика)

## Календарь
- Подсветка «сегодня», легенда статусов
- Иконки на слотах: ✓ проведён, ↻ перенесён, ✕ отменён
- Проведённые — лавандовый фон

## Списки занятий
- Строки с аватаром, курсом, датой, иконкой статуса

## Вход
- `redirectTo` + `window.location.assign` (без ошибки server action)
- try/catch при недоступном Supabase
