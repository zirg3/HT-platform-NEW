# v0.07-4 — список занятий и финальный UX

## Что сделано

- **AgendaWithDialog**: клик по строке в «Предстоящие / Прошедшие» открывает карточку урока (отмена, ссылка, ДЗ).
- **AgendaLessonLists**: интерактивные строки с focus/hover; ссылка на урок не всплывает при клике по строке.
- **SchedulePage**: подсказки внизу обновлены под календарь + список.
- **RoleNav**: горизонтальная прокрутка на узком экране.
- **Админ-формы**: `bg-background` у select; `aria-busy` на кнопках удаления/отмены в диалоге урока.
